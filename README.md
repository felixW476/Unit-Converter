# Unit-Converter

## Overview
Build a simple web page that will have different sections for different units of measurement. The user can input a value to convert, select the units to convert from and to, and view the converted value.

- The user can input a value to convert.
- The user can select the units to convert from and to.
- The user can view the converted value.
- The user can convert between different units of measurement like length, weight, and temperature.

## Features
- **Length:** millimeter, centimeter, meter, kilometer, inch, foot, yard, mile
- **Weight:** milligram, gram, kilogram, ounce, pound.
- **Temperature:** Celsius, Fahrenheit, Kelvin.

## Software/Implementation Requirement
- Tomcat 9
- Java
- JSP
- Servlet
- VS code

## How to Run

### 1. Clone the repository
```bash
git clone https://github.com/felixW476/Unit-Converter.git
cd Unit-Converter
```
### 2. Run Tomcat
```bash
Open your command prompt
C: or D: cd \apache-tomcat-9.0.113-windows-x64\apache-tomcat-9.0.113\bin
startup.bat
```
### 3. Open the webpage application
Open web browser
```bash
http://localhost:8080/unitconverter/index.jsp
```

## Troubleshooting 
If the *.class files do not compile, try Ctrl + Shift B to build in VS Code or restart your PC.
